========================================================================
    动态链接库：iConnectModule 项目概述
========================================================================

此模块为Pss框架下的模块。用于实现信息网关功能。
功能：
    业务分为N各区域。每个区域有M个应用，通过一个网络网关统一管理应用之间或者
不同区域之间的网络通讯。
    应用分四种
    点对点应用，只允许和相同的应用进行通讯
    APP应用，通过认证间接和资源应用通讯
    资源应用，返回数据给请求的APP应用
    认证应用，管理所有对资源的请求，是否允许
    
    服务端只关注数据转发。处理应用客户端的连接，注册，转发数据，同步在线状态。
    服务器有两种ID，数字型的id，字符串型的StationID，可以互相转换。

因为数据包按80k分包
所以需要将pss的main.xml 里的配置RecvBuffSize 设为 大于80k，比如 "102400"
